- Tray/minimized efficiency
- Hidden-surface suppression
- Real-world user experience

- [ ] **Battery %instability**: No aggressive telemetry cadence during idle
- [ ] **OSD duplication**: OSD updates via single cadence, no duplicate renders

## Rollback Criteria

If ANY of the following are observed, recommend **rollback or escalation**:
- Thermal idle > 5°C above v3.6.1 baseline
- Dispatcher amplification > 3:1 during normal use
- Dashboard responsiveness < 30fps during heavy load
- Hotkey latency > 500ms to applied state
- Memory growth > 50MB/hour during idle
- Fan ramping becomes erratic or chattering
- Any crash or hang during validation session
- Any mode drift or unexpected state changes
